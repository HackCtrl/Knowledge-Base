## **ШАГ 8: ПРАКТИКУМ - СОЗДАНИЕ ФАЙЛОВ**

***8.1. Создадим тестовый файл*** 
- `sudo touch /mnt/data/testfile`

***8.2. Проверим создание*** 
- `ls -la /mnt/data/`

***8.3. Создадим доп файлы для экспериментов*** 

```
# Создаем текстовый файл
echo "Этот файл создан для практической работы по LVM с GPT разделами" | sudo tee /mnt/data/practic.txt

# Создаем файл для снепшотов
echo "Файл для экспериментов со снепшотами LVM" | sudo tee /mnt/data/snapshot_test.txt

# Создаем директорию
sudo mkdir /mnt/data/test_dir

# Создаем несколько файлов в директории
for i in {1..3}; do
    echo "Файл $i для тестирования LVM" | sudo tee /mnt/data/test_dir/file$i.txt
done
```

***8.4. Проверим все созданные файлы*** 

```
# Смотрим содержимое
ls -la /mnt/data/

# Смотрим содержимое директории
ls -la /mnt/data/test_dir/

# Просматриваем содержимое файлов
sudo cat /mnt/data/practic.txt
sudo cat /mnt/data/snapshot_test.txt
sudo cat /mnt/data/test_dir/file1.txt
```

## **ШАГ 9: РАБОТА СО ВТОРЫМ РАЗДЕЛОМ (/dev/sdb2)**

***9.1. Проверим второй раздел*** 

```
# Посмотрим на второй раздел
sudo fdisk -l /dev/sdb | grep sdb2

# Проверим тип данных
sudo file -s /dev/sdb2
```

![[Pasted image 20260103110424.png]]

***9.2. Создадим ФС напрямую на втором разделе*** 

```
# Форматируем второй раздел в ext4
sudo mkfs.ext4 /dev/sdb2

# Создаем точку монтирования
sudo mkdir /mnt/data2

# Монтируем
sudo mount /dev/sdb2 /mnt/data2

# Проверяем
df -hT /mnt/data2
```

## **ШАГ 10: ПОЛНАЯ ПРОВЕРКА ВСЕЙ СТРУКТУРЫ**

***10.1. Запускаем комплексную проверку*** 

```
echo "=== 1. СТРУКТУРА ДИСКОВ И РАЗДЕЛОВ ==="
lsblk

echo -e "\n=== 2. GPT ТАБЛИЦА РАЗДЕЛОВ ==="
sudo fdisk -l /dev/sdb | grep -A20 "Disk /dev/sdb"

echo -e "\n=== 3. LVM СТРУКТУРА ==="
echo "Физические тома (PV):"
sudo pvs
echo -e "\nГруппы томов (VG):"
sudo vgs
echo -e "\nЛогические тома (LV):"
sudo lvs

echo -e "\n=== 4. ФАЙЛОВЫЕ СИСТЕМЫ ==="
df -hT | grep -E "(Filesystem|data)"

echo -e "\n=== 5. ПУТИ К УСТРОЙСТВАМ ==="
ls -la /dev/mapper/ | grep data
ls -la /dev/data/

echo -e "\n=== 6. СОДЕРЖИМОЕ /mnt/data ==="
ls -la /mnt/data/
```

***10.2. Проверка UUID и меток***

```
# Проверяем UUID файловых систем
sudo blkid /dev/sdb1
sudo blkid /dev/sdb2
sudo blkid /dev/data/lv
```
