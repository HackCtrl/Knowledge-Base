Реляционные БД управляются при помощи языка программирования SQL (язык структурированных запросов)

***SQL*** - Это декларативный язык (то есть программист пишет, что должно получиться, а система сама решает, как это должно получиться)

***Диалект*** - это набор дополнительных команд, расширяющих стандартные возможности языка SQL
