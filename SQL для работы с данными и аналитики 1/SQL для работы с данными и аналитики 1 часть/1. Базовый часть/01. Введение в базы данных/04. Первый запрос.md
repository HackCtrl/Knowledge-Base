Запросы в языке SQL пишут с помощью операторов. Операторы описывают действия над данными (выбрать, удалить или добавить) 

SELECT - оператор, с помощью которого описываем, что выгружать из БД 
FROM - оператор, с помощью которого описывают, откуда выгружать данные 

```
SELECT 'на экран отображается число', 5; 
```
