### Задание

Из таблицы `users` требуется вывести информацию о пользователях, у которых либо не указана фамилия либо количество очков опыта строго больше 100**:**

1. Идентификатор
2. Логин
3. Количество очков опыта на платформе

### Столбцы в результате

1. id
2. username
3. score

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте по возрастанию поля `id`.

### ***Решение*** 

```
select id, username, score 
from users 
where 
    last_name is null
    or
    score > 100
order by id; 
```
