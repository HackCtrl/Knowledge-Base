### Задание

Из таблицы `users` требуется вывести информацию об активных пользователях (`is_active = 1`), которые при этом либо имеют количество очков опыта строго больше 500, либо относятся к компании с `id = 7`:

1. Идентификатор
2. Логин
3. Идентификатор компании
4. Количество очков опыта на платформе

### Столбцы в результате

1. id
2. username
3. company_id
4. score

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте по возрастанию поля `id`

### ***Решение***

```
select id, username, company_id, score
from users 
where 
    is_active = 1
    and
    (
    score > 500 
    or
    company_id = 7 
    )
order by id;
```
