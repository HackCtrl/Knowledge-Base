### Задание

Из таблицы `users` требуется вывести следующую информацию о пользователях**:**

1. Идентификатор
2. Логин
3. Почта
4. Дата и время регистрации

### Столбцы в результате

1. id
2. username
3. email
4. date_joined

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте сначала по убыванию поля `date_joined`, а затем по возрастанию поля `id`.

### ***Решение*** 

```
select id, username, email, date_joined from users
order by date_joined desc, id; 
```
