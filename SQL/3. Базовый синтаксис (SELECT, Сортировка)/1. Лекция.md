***2 неотъемлемые части SQL запроса:*** 
1. Что мы хотим взять 
2. Откуда мы хотим взять 

`SELECT ...` - перечисление столбцов, которые мы хотим получить 
`FROM ...` - из какой таблицы мы хотим эти столбцы получить 

***Пример запроса:*** 
```
select * from users;
```

`*` - говорит о том, чтоб выбрать все столбцы 

*То сесть запрос читается так - выбрать все столбцы из таблицы users* 

***Алиас*** - это короткой имя, которое мы даем какому либо объекту, для того, чтоб потом было удобно к нему обращаться 

***Например:***
- Вместо таблицы users мы можем обращаться по алиасу `u` 

***Пример запроса:***
```
select
	u.username login,
	u.date_joined as created_at
from users u;
```

В результате мы обратились к двум столбацам и вывели их `username` и `date_joined` их таблички `users`. Так же мы переназываем столбец `username` на `login`, а `date_joined` на `created_at`.  `as` у нас используется для переназывания столбца, но можно писать и без него, как у нас написано на 2 и 4 строчке 

***Соскировка*** - это то, в каком порядке мы хотим выводить строки 

Столбцы мы сортируем так, то есть что первым напишем так и будет выводиться, то есть мы выше написали первым `login`, а потом `created_at`, а если наоборот, то в каком порядке написали, так и выведет 

А вот если строки, то тут явно нужно указывать сортировку и по какому столбцу мы сортируем 

***Пример запроса:*** 
```
select * from users u order by date_joined asc;
```

- Хотим вывести все столбцы из таблицы users, отсортировав наш результат по полю `date_joined` 
- `asc` - по возрастанию 
- `desc` - по убыванию 

*У оператора `order by` по умолчанию идет сортировка по возрастанию* 

***Пример запроса:*** 
```
select * from users u order by is_active, date_joined desc;
```

- `is_active` - по умолчанию у нас сортируется по возрастанию 
- `date_joined` - сортируется по уменьшению из за функции `desc`
