### Задание

Из таблицы `users` требуется вывести информацию о пользователях, которые имеют домен электронной почты `bk.ru`.

### Столбцы в результате

1. id
2. username
3. email

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте по возрастанию поля `id`
### ***Решение***

```
select 
    id,
    username, 
    email
from users
where email like '%bk.ru%'
order by id; 
```
