### Задание

Из таблицы `users` требуется вывести следующую информацию о пользователях**:**

1. Идентификатор
2. Логин
3. Почта
4. Домен электронной почты (вычисляется как часть электронной почты после символа `@`)

### Столбцы в результате

1. id
2. username
3. email
4. domain

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте по возрастанию поля `id`
### ***Решение*** 

```
select
    id, 
    username, 
    email, 
    right(email, length(email) - position('@' in email)) as domain
from users
order by id; 
```
