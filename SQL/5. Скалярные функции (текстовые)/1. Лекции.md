***Скалярные функции*** - это функции, которые мы применяем к набору данных, можно применять к столбцу или к одному значению, чтоб как то их преообразовать (например работа с датой и временем, там допустим, чтоб выбрать только месяц и так далее)

`lower()` - для перевода в нижний регистр 

```
select lower('ПриВЕт');
```

Строки в Postgres задаются одинарными кавычками `''`. В двойных кавычках "" указываются объекты БД (название столбцов, строк и так далее)

```
select 
	lower('ПриВЕт') as "lower_case", 
	'ПриВЕт' as init_phrase; 
```

`upper()` - переводит в верхний регистр
`initcap()` - переводит все слова в предложении, чтобы первая буква была большая 

```
select name,
	lower(name),
	upper(name),
	initcap(name)
from problem p;
```

*Допустим мы пишем такой код:*
```
select *
from users
where username = 'Девелопер';
```

Тут у нас Девелопер написан и мы сразу угадали, как он в таблице пишется, а если он будет написан в другом регистре, то придется гадать 

Есть варианты решить данную задачу

1. Перевести в нижний регистр и тем самым найти нужный нам параметр 
```
select *
from users 
where lower(username) = 'девелопер'; 
```

2. Точно так же с функцией `upper()`: 
```
select *
from users 
where upper(username) = ДЕВЕЛОПЕР; 
```

`like` - оператор, который ищет совпадения по шаблону 

```
select *
from problem p 
where name like '%new%'; 
```

То есть тут мы при помощи оператор `like` задаем шаблон `'%new%',` который говорит, что поле должно начинаться со слова в шаблоне

`%%` - говорят нам о том, что справа или слева могут быть любые символы, в том числе пустота

А что если нам нужно найти именно те, что встречается в строках, то есть в определенном регистре 

1. Это так же привести все в нижний регистр 
```
select *
from problem p 
where lower(name) like '%new%'; 
```

`ilike` - регистр независимый поиск 

```
select *
from problem p 
where name ilike '%new%'; 
```

`length()` - функция для определения длины строки 

```
select name, length(name)
from problem p; 
```

`left()` - используется для извлечения заданного количества символов с начала строки 
`right()` - извлекает и возвращает заданное количество символов с конца строки 
`position()` - используется для поиска начальной позиции подстроки в строке 

```
select email, right(email, position('@' in email))
from users;
```
