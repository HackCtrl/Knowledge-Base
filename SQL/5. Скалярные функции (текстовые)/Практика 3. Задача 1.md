### Задание

Из таблицы `users` требуется вывести следующую информацию о пользователях:

1. Идентификатор
2. Логин
3. Имя
4. Фамилия
5. Имя в нижнем регистре
6. Фамилия в верхнем регистре
7. Длина логина

### Столбцы в результате

1. id
2. username
3. first_name
4. last_name
5. lower_first_name
6. upper_last_name
7. length_username

**Важно**: Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте сначала по убыванию `length_username`, затем по возрастанию `id`
### ***Решение***

```
select 
    id, 
    username,
    first_name, 
    last_name,
    lower (first_name) as lower_first_name, 
    upper(last_name) as upper_last_name, 
    length(username) as length_username
from users
order by length_username desc, id; 
```
