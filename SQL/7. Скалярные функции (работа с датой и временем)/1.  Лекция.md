`to_char()` - функция, которая переводит дату и время в строку по нужному формату 

```
select
	username,
	date_joined,
	to_char(date_joined, 'YYYY-MM') as ym
from users
```

```
select
	username,
	date_joined,
	to_char(date_joined, 'YYYY-MM') as ym
from users
where to_char(date_joined, 'YYYY-MM') = '2021-11'
```

`date_trunc()` - функция, которая обрезает дату или время до заданного уровня точности 

```
select
	username,
	date_joined,
	to_char(date_joined, 'YYYY-MM') as ym,
	date_trunc('month', date_joined) as ym2
from users
```

```
select
	username,
	date_joined,
	to_char(date_joined, 'YYYY-MM') as ym,
	date_trunc('month', date_joined) as ym2,
	to_char(date_joined, 'YYYY-MM-DD') as ym3,
	date_trunc('day', date_joined) as ymd4
from users
```

`make_date()` - функция, которая собирает полноценную дату из отдельных числовых компонентов (года, месяца и дня)

```
select *, make_date(2027, 5, 5)
from users
```

***Так работает вычитание дат:*** 
```
select
	date_joined,
	make_date(2027, 5, 5) - date_joined as diff
from users
```

`now()` - функция, которая возвращает текущую дату и точное время 

```
select
	date_joined,
	make_date(2027, 5, 5) - date_joined as diff,
	now() - date_joined as diff2,
from users
```

`extract()` - функция, которая извлекает заданную часть (день, месяц, год, час) из значения даты или времени 

```
select
	date_joined,
	make_date(2027, 5, 5) - date_joined as diff,
	now() - date_joined as diff2,
	extract(days from make_date(2027, 5, 5) - date_joined) as diff_days
from users
```

***Чтоб посчитать точное количество часов, сколько прошло:***
```
select
	date_joined,
	make_date(2027, 5, 5) - date_joined as diff,
	now() - date_joined as diff2,
	extract(days from make_date(2027, 5, 5) - date_joined) * 24 + extract(hours from make_date(2027, 5, 5) - date_joined) as diff_days
from users
```

`INTERVAL` - специальный типа данных, который представляет собой отрезок или промежуток времени 

```
select
	date_joined,
	make_date(2027, 5, 5) - date_joined as diff,
	now() - date_joined as diff2,
	extract(days from make_date(2027, 5, 5) - date_joined) * 24 + extract(hours from make_date(2027, 5, 5) - date_joined) as diff_days,
	date_joined + interval '3 days'
from users
```
