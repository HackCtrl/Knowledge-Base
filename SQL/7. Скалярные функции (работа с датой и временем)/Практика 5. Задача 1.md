### Задание

Из таблицы `users` требуется вывести следующую информацию о пользователях:

1. Идентификатор
2. Логин
3. Почта
4. Дата и время регистрации
5. Дата и время регистрации, округленная до первого числа месяца (например, `2022-04-21 12:20:00.000202` —> `2022-04-01 00:00:00`)

### Столбцы в результате

1. id
2. username
3. email
4. date_joined
5. registration_month_start

**Важно**: Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте сначала по убыванию поля `date_joined`, а затем по возрастанию поля `id`

### ***Решение***

```
select 
    id, 
    username, 
    email, 
    date_joined, 
    date_trunc('month', date_joined) as registration_month_start
from users
order by date_joined desc, id asc;
```
