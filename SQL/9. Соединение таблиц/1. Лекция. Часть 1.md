`INNER JOIN` - внутренее соединение, связывает строки из двух таблиц. Он выводит только те строки, для которых нашлось совпадение по условию в обеих таблицах. Если совпадений нет, строка в результат не попадает 

`LEFT JOIN` - левое соединение, соединяет две таблицы. Он берет все строки из первой (левой) таблицы и добавляет к ним совпадающие строки из второй (правой) таблицы. Если для левой строки нет пары справа, база данных подставит `NULL` (пустое значение)

`RIGHT JOIN` - соединяет все таблицы и сохраняет все строки из правой таблицы. Если в левой таблице нет пары для строки справа, в её полях записывается `NULL` (пустое значение)

`FULL JOIN` - объединяет все данные из двух таблиц. Он сохраняет строки, даже если для них нет совпадений. Отсутствующие значения заполняются словом `NULL`(пустота). Он работает как сумма `LEFT JOIN` и `RIGHT JOIN`
