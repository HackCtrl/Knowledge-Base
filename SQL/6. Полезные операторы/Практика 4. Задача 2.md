
### Задание

Из таблицы `users` требуется вывести следующую информацию о пользователях**:**

1. Идентификатор
2. Логин
3. Имя
4. Фамилия
5. Приветственное имя пользователя:
6. если есть имя, то вывести имя пользователя,
7. если имени нет, но есть фамилия, то вывести фамилию,
8. если отсутствуют как имя, так и фамилия, то вывести `Дорогой друг`

### Столбцы в результате

1. id
2. username
3. first_name
4. last_name
5. display_name

**Важно:** Обратите внимание, что название столбцов в вашем ответе должно в точности совпадать с условием.

### Сортировка

Результат отсортируйте по возрастанию поля `id`

### ***Решение***

```
select 
    id, 
    username, 
    first_name, 
    last_name, 
    case 
        when first_name is not null then first_name
        when first_name is not null then last_name
        else 'Дорогой друг'
    end as display_name
from users
order by id; 
```
