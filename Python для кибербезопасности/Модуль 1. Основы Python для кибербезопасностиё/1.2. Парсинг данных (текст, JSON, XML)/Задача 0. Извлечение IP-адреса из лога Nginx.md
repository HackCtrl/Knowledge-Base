Веб-серверы, такие как Nginx, записывают информацию о каждом запросе в лог-файлы. Анализ этих логов критически важен для отслеживания посещаемости, поиска ошибок и обнаружения подозрительной активности, например, попыток взлома или DDoS-атак.

Стандартный формат лога Nginx обычно начинается с IP-адреса клиента, сделавшего запрос. Ваша задача — извлечь этот IP-адрес из одной строки лога.

**Цель**

Написать программу на Python, которая:

1. Получает на вход **одну строку** — запись из лога Nginx (через стандартный ввод).
2. Извлекает из этой строки самую первую часть — IP-адрес клиента (он идет до первого пробела).
3. Выводит на экран **только этот IP-адрес**.

**Пример ввода:**

192.168.5.12 - - [11/Oct/2023:14:25:01 +0300] "GET /images/logo.png HTTP/1.1" 200 1578 "http://example.com/" "Mozilla/5.0 ..."

**Пример вывода для этого ввода:**

192.168.5.12

**Другой пример ввода:**

203.0.113.5 - user1 [11/Oct/2023:14:26:30 +0300] "POST /login HTTP/1.1" 401 50 "-" "curl/7.68.0"

**Пример вывода для этого ввода:**

203.0.113.5

Эта задача научит вас разбивать строки на части — основному методу парсинга простого текстового формата.
