Описание задачи 

Большинство систем безопасности требуют аутентификации пользователя — подтверждения личности. Самый простой способ — запрос имени пользователя (логина) и пароля. Сейчас мы сосредоточимся на первом шаге: получении имени пользователя.

Ваша задача — написать программу, которая будет "спрашивать" у пользователя его имя и, получив его, выводить персональное приветствие.

**Цель** 
Написать программу на Python, которая:

1. Запрашивает у пользователя ввод имени (логина).
2. Считывает введенное имя.
3. Выводит на экран приветственное сообщение в формате: `Welcome, [введенное_имя]!` (где `[введенное_имя]` — это то, что пользователь ввел).

Например, если пользователь введет `admin`, программа должна вывести `Welcome, admin!`.

Это важный шаг к созданию интерактивных программ, которые могут реагировать на действия пользователя. В системах безопасности это используется повсеместно: от входа в систему до запроса подтверждений на выполнение опасных операций.

`input()` - чтобы программа могла получать информацию от пользователя во время ее выполнения 

**Как работает `input()`?**
1. Когда Python встречает `input()`, выполнение программы приостанавливается.
2. Она ждет, пока пользователь что-нибудь напечатает на клавиатуре и нажмет Enter.
3. Введенный пользователем текст возвращается функцией `input()` в виде **строки**(`str`).

**Объединение строк для вывода**

Нам нужно вывести сообщение, состоящее из фиксированной части (`"Welcome, "`), введенного имени и еще одной фиксированной части (`"!"`). Есть несколько способов это сделать:

1. **Конкатенация (сложение строк).** Можно "склеить" строки с помощью оператора `+`. `greeting = "Welcome, " + user_login + "!"` `print(greeting)`
2. **f-строки (форматированные строки - рекомендуемый способ).** Это современный и удобный метод. Строка начинается с буквы `f` перед кавычками, а имена переменных, значения которых нужно подставить, указываются внутри фигурных скобок `{}`. `greeting = f"Welcome, {user_login}!"` `print(greeting)`
