В программировании часто используются так называемые "флаги" — переменные, которые могут находиться только в двух состояниях: "включено" или "выключено", "истина" или "ложь". Они помогают отслеживать состояние системы или процесса.

Например, в системе безопасности может быть флаг, показывающий, установлено ли защищенное соединение (`True`) или нет (`False`).

Ваша задача — создать такой флаг и установить его в состояние "безопасно".

**Цель**

Написать программу на Python, которая:

1. Создаст переменную с именем `is_secure`.
2. Присвоит этой переменной значение "истина" (`True`).
3. Выведет значение этой переменной на экран.

**Входные данные**

Для этой задачи входные данные не требуются.

**Ожидаемый вывод:**

True

Эта задача познакомит вас с булевым типом данных — основой логики в программировании.

### ***Теория***

Кроме строк (`str`) и целых чисел (`int`), в Python есть еще один базовый тип данных — **булев** или **логический** тип (`bool`)

**Два возможных значения:** 
- `True` - истина, правда, да, включено 
- `False` - ложь, неправда, нет, выключено 

**Создание булевой переменной**
- `connection_ok = True`
- `access_denied = False`

**Результат сравнения — булево значение**
- `result = (5 == 5) # result будет True`

### ***Решение*** 

```
is_secure = True 
print(is_secure)
```
