Описание задачи 

В сетевой безопасности часто нужно проверять, открыты ли на сервере определенные порты. Некоторые порты известны как часто используемые для стандартных служб (например, порт 80 для веб-серверов), а другие могут быть связаны с потенциальными уязвимостями или вредоносным ПО.

Представим очень упрощенную ситуацию: у нас есть список "опасных" портов, и мы хотим проверить, совпадает ли номер проверяемого порта с одним из этих опасных номеров.

**Цель**
Написать программу на Python, которая:

1. Получает на вход **два целых числа**, каждое на новой строке:
    - Первое число: номер порта, который нужно проверить (`**port_to_check**`).
    - Второе число: номер "опасного" порта (`**dangerous_port**`).
2. Сравнивает эти два числа на равенство.
3. Выводит на экран слово `True`, если номера портов совпадают.
4. Выводит на экран слово `False`, если номера портов не совпадают.

**Пример ввода:**

80
23

**Пример вывода для этого ввода:**

False

**Другой пример ввода:**

1337
1337

**Пример вывода для этого ввода:**

True

Эта задача научит вас сравнивать числовые значения и работать с логическими результатами (истина/ложь), что является основой для принятия решений в твоих программах.
### ***Теория***

== - оператор сравнения на равенство 

В нашем случае я использовал конструкцию `if else`, но можно было сделать и по другому используя просто оператор сравнение на равенство `are_equal = (port1_num == port2_num)`

***Что я усвоил***
- Я сравнил проверяемый порт с одним из опасных портов
### ***Решение***

```
# put your python code here

port1 = int(input())
port2 = int(input())

if port1 == port2: 
    print('True')
else: 
    print('False')
```
