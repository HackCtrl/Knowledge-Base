Описание задачи 

Одним из базовых требований к надежному паролю является его достаточная длина. Чем длиннее пароль, тем сложнее его подобрать злоумышленнику. Системы аутентификации часто проверяют длину пароля при его создании или смене.

Ваша задача — написать программу, которая имитирует эту проверку: она будет считывать введенный пароль и сообщать его длину.

**Цель**
Написать программу на Python, которая:

1. Запрашивает у пользователя ввод пароля (любая строка символов).
2. Определяет количество символов в введенной строке (ее длину).
3. Выводит на экран только это число (длину пароля).

**Пример:**
- Если пользователь вводит `qwerty`, программа должна вывести `6`.
- Если пользователь вводит `MySecureP@ssw0rd`, программа должна вывести `17`.
- Если пользователь вводит пустую строку (просто нажимает Enter), программа должна вывести `0`.

Эта задача научит вас измерять "размер" текстовых данных, что полезно не только для паролей, но и для анализа логов, сообщений и любых других текстовых фрагментов.
### ***Теория*** 

`len()` - функция, чтобы узнать сколько символов содержится в строке

***Что я усвоил***
- Я научился подсчитывать количество символов в пароле веденным пользователем для подтверждения его актуальности и безопасности 
### ***Решение***

