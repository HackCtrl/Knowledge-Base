`print()` - используется в python, чтобы вывести что-либо на экран (в стандартный поток вывода, обычно это консоль)

Представьте, что вы работаете с системой мониторинга безопасности. Одна из ее базовых функций — выводить статусное сообщение, когда все работает в штатном режиме. Ваша первая задача — научить программу выводить на экран консоли определенное сообщение.

**Цель** 
Написать программу на Python, которая выведет на экран строку: `System Alert: All systems nominal.`

Это самое простое, но и самое важное действие в программировании — умение "сказать" что-то пользователю или другой системе. В кибербезопасности это постоянно используется для вывода логов, сообщений об ошибках, статусов проверки и т.д.

***Что я усвоил***
- Мы научили программу выводить статусное сообщение, когда все работает в штатном режиме
### ***Решение***

```
print("System Alert: All systems nominal.")
```
