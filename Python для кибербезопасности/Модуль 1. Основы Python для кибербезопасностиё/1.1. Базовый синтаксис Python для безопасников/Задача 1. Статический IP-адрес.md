***Переменная*** - это как именованный контейнер для ячейки памяти, куда можно положить какие-то данные. У переменной есть имя (идентификатор) и значение

В сетевых настройках часто требуется указать статический (неизменяемый) IP-адрес для устройства, например, для сервера или важного сетевого компонента. Это позволяет другим системам всегда находить его по одному и тому же адресу.

В этой задаче мы научимся "запоминать" такой IP-адрес в программе с помощью переменной.

**Цель** 
Написать программу на Python, которая:

1. Создаст переменную.
2. Присвоит ей значение IP-адреса `"192.168.1.100"` (как строку).
3. Выведет значение этой переменной на экран.

Умение хранить и использовать данные — фундаментальная часть программирования. IP-адреса, имена хостов, ключи шифрования — всё это данные, с которыми специалисту по кибербезопасности приходится работать постоянно.

- Я научился запоминать статический (неизменяемый) IP-адрес для устройства в переменной для дальнейшего обращения к ней
