Системы безопасности часто отслеживают количество неудачных попыток входа в учетную запись. Если число попыток превышает определенный порог, система может временно заблокировать аккаунт или потребовать дополнительную проверку. Это помогает защититься от подбора паролей (брутфорса).

В этой задаче мы смоделируем самый простой элемент такого счетчика.

**Цель** 
Написать программу на Python, которая:

1. Получает на вход **одно целое число** — текущее количество неудачных попыток входа.
2. Увеличивает это число на 1 (имитируя еще одну неудачную попытку).
3. Выводит **новое** (увеличенное) значение счетчика на экран.

**Пример:**
- Если на вход подается число `5`, программа должна вывести `6`.
- Если на вход подается число `0`, программа должна вывести `1`.

Эта задача познакомит вас с работой с целыми числами и выполнением базовых арифметических операций — ключевыми навыками для анализа данных и автоматизации задач в кибербезопасности.
### ***Теория***

`int` - это целые числа без дробной части 

Так как input() по умолчанию принимает текст, нам нужно преобразовать в числа при помощи `int(input())`

***Что я усвоил***
- Мы научились принимать на ввод число и прибавлять к нему и выводить в консоль, это сделана для подсчета неудачных попыток авторизации для предотвращения брутфорс атаки 
### ***Решение*** 

```
# put your python code here

inter = int(input())
name = inter + 1
print(name)
```
