Описание задачи 

Системы безопасности часто логируют (записывают) длительность пользовательских сессий. Иногда это время записывается в минутах, но для детального анализа или сравнения может потребоваться перевести его в секунды.

Ваша задача — написать программу, которая выполняет это простое преобразование.

**Цель** 
Написать программу на Python, которая:

1. Получает на вход **одно целое число** — длительность сессии в минутах.
2. Преобразует это время в секунды (путем умножения на 60).
3. Выводит на экран **полученное количество секунд** (тоже целое число).

**Пример ввода:**

15

**Пример вывода для этого ввода:**

900

**Другой пример ввода:**

1

**Пример вывода для этого ввода:**

60

**Еще пример ввода:**

0

**Пример вывода для этого ввода:**

0

Эта задача закрепит ваши навыки работы с вводом данных, преобразованием типов и выполнением арифметических операций.
### ***Теория*** 

`int()` - преобразование в целые числа 

`*` - оператор умножения

***Что я усвоил*** 
- Научился переводить минуты в секунды для детального анализа или сравнения 
### ***Решение***

```
# put your python code here

time = int(input())
result = time * 60
print(result)
```
