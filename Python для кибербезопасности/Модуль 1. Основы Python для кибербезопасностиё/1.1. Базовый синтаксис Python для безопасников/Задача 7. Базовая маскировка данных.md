Описание задачи 

При работе с конфиденциальной информацией, такой как номера кредитных карт, API-ключи или пароли, часто не требуется (и небезопасно!) отображать ее целиком. Вместо этого показывают только часть данных, а остальное скрывают за маской (например, звездочками). Это называется **маскировкой данных**.

Ваша задача — реализовать простейший механизм маскировки.

**Цель**
Написать программу на Python, которая:

1. Получает на вход **одну строку** произвольной длины (например, API-ключ).
2. Берет **первые 4 символа** этой строки.
3. Добавляет к этим символам строку `"****"`.
4. Выводит **результирующую строку** (первые 4 символа + маска) на экран.

**Важно!** Если исходная строка короче 4 символов, нужно взять все символы, которые есть, и добавить к ним `"****"`.

**Пример ввода:**

sk-abcdefghijklmnopqrstuvwxyz12345

**Пример вывода для этого ввода:**

sk-a****

**Другой пример ввода (короче 4 символов):**

abc

**Пример вывода для этого ввода:**

abc****

Эта задача научит вас извлекать части строк и объединять их — полезные навыки для обработки любых текстовых данных.

### ***Теория***

**Срезы строк (Slicing)**

Срезы используются для того, чтобы получить не всю строку, а только ее часть. Они указываются в квадратных скобках `[]` после имени переменной строки с помощью `:`

**Формат:** 
- `строка[начало:конец]`

- ***Начало*** - индекс (номер) символа, с которого начинается срез (включительно). Нумерация символов в строке начинается с 0 
- ***Конец*** - индекс символа, на котором срез заканчивается (не включительно)

**Полезные варианты срезов:**
- `строка[:N]` - взять символы с самого начала до индекса N (не включая сам N). То что нужно для получения первых N символов 
- `строка[N:]` - взять символы от индекса N (включительно) до самого конца строки 

***Пример:*** 

```
api_key = "sk-abcdef12345"

first_four = api_key[:4] # Берем символы с индексами 0, 1, 2, 3

print(first_four) # Выведет: sk-a

short_key = "abc"

first_four_short = short_key[:4] # Если символов меньше 4, возьмет все, что есть

print(first_four_short) # Выведет: abc
```

**Конкатенация строк (повторим)**

Чтобы склеить две или более строк в одну, используется оператор сложения `+`
- `part1 = "Hello"`
- `part2 = ", World!"`
- `greeting = part1 + part2 # Склеиваем две части`
- `print(greeting) # Выведет: Hello, World!`

### ***Решение***

```
getting_the_data = input()
first_four = getting_the_data[:4]
greeting = first_four + "****"
print(greeting)
```
