### ***Виды связей*** 

***Simplex*** - односторонняя связь (имеется один передачек и несколько приемников, который получают один и тот же сигнал)

![[Pasted image 20260407205200.png]]

***Hulf-duplex*** - двусторонняя связь, при которой в один момент времени может передавать только одно устройство (все устройства будут передавать по очереди, например рации)

![[Pasted image 20260407205342.png]]

***Full-duplex или duplex*** - двусторонняя связь, при которой оба устройства могут вести передачу одновременно 

![[Pasted image 20260407205510.png]]
### ***Методы передачи данных*** 

***Unicast*** - передача данных единственному адресату 
![[Pasted image 20260407205730.png]]
***Broadcast*** - широковещательная передача данных всем адресатам \
![[Pasted image 20260407205716.png]]
### ***Виды коммутации*** 

***Коммутация каналом*** - в сети с коммутацией каналов между двумя кончеными устройствами устанавливается физический канал 

![[Pasted image 20260407205907.png]]

***Коммутация пакетов*** - в сети с коммутацией пакетов информации от каждого устройства делится на небольшие пакеты, и данные передаются по одним и тем же физическими каналам 

![[Pasted image 20260407210030.png]]

***WAN (Wide Area Network)*** - глобальная вычислительная сеть 
***LAN (Local Area Networl)*** - локальные сети 
***MAN (Metropolitan Area Network)*** - сеть городского масштаба
