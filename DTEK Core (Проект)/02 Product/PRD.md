# PRD (Product Requirements Document)

`Проект: DTEK Core`
`Версия: 1.0`
`Статус: Draft`

# 1. Описание продукта

DTEK Core — платформа управления цифровым доверием организации.

Система позволяет создавать цифровую модель организации, управлять объектами, их взаимосвязями, рисками и уровнем доверия.

# 2. Цели MVP

**Первая версия продукта должна позволять:**
- создавать организации;
- управлять пользователями;
- создавать объекты;
- формировать цифровые паспорта доверия;
- рассчитывать базовый Trust Score;
- визуализировать граф доверия;
- вести реестр рисков.

# 3. Пользователи системы

## Owner

`Владелец организации.`

**Права:**
- полный доступ к организации;
- управление пользователями;
- просмотр всех данных.

## Security Officer

`Специалист по ИБ.`

**Права:**
- управление объектами;
- управление рисками;
- управление паспортами доверия.

## Administrator

`Системный администратор.`

**Права:**
- управление инфраструктурными объектами.

## Viewer

`Только просмотр данных.`

# 5. Ограничения MVP

**Не входят:**
- AI Assistant;
- Marketplace;
- Отраслевые конфигурации;
- Enterprise Runtime;
- Интеграции со сторонними системами.

# 6. Критерий готовности MVP

**Пользователь может:**
1. зарегистрироваться;
2. создать организацию;
3. добавить объект;
4. получить паспорт доверия;
5. увидеть Trust Score;
6. увидеть связи объекта на графе;
7. зарегистрировать риск.
