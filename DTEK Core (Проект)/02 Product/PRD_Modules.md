# PRD Modules

## Модуль 1. Организации

### Назначение

Позволяет пользователю создавать и управлять организациями внутри системы.

### Основные функции

- создание организации;
- редактирование организации;
- просмотр информации об организации;
- удаление организации;
- приглашение пользователей.

### Поля организации

- ID;
- название;
- описание;
- отрасль;
- размер организации;
- дата создания;
- владелец.

### Роли

`Owner`
`Security Officer`
`Viewer`

### Критерий готовности

Пользователь может создать организацию и управлять её данными.

## Модуль 2. Пользователи

### Назначение

Управление пользователями организации.

### Основные функции

- приглашение пользователя;
- удаление пользователя;
- назначение ролей;
- просмотр профиля.

### Поля пользователя

- ID;
- имя;
- email;
- роль;
- дата регистрации.

### Критерий готовности

Пользователь может пригласить нового участника и назначить ему роль.

## Модуль 3. Объекты

### Назначение

Хранение цифровых объектов организации.

### Типы объектов MVP

- сотрудник;
- сервер;
- рабочая станция;
- ноутбук;
- приложение;
- база данных;
- сетевое устройство.

### Поля объекта

- ID;
- название;
- тип;
- владелец;
- критичность;
- статус;
- дата создания.

### Основные функции

- создание объекта;
- редактирование объекта;
- удаление объекта;
- просмотр объекта.

### Критерий готовности

Пользователь может создать объект и сохранить его в системе.
