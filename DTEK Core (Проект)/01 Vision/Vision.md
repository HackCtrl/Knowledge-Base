# DTEK Core Vision

## О проекте

DTEK Core — это платформа управления цифровым доверием организации.

Проект создается как единая экосистема для управления объектами организации, их взаимосвязями, рисками и уровнем доверия.

Основная идея заключается в том, что любой объект организации должен иметь цифровой паспорт доверия и быть частью единой модели цифрового доверия компании.

DTEK Core не является очередным средством защиты информации, SIEM, DLP или EDR.

DTEK Core выступает надстройкой над существующими средствами защиты и формирует единую модель цифрового доверия организации.

## Проблема

**Сегодня организации используют множество разрозненных решений:**
- антивирусы;
- средства управления доступом;
- системы мониторинга;
- системы управления уязвимостями;
- средства учета активов.

Однако отсутствует единая система, позволяющая видеть цифровое состояние организации как единого объекта.

**Руководство компании не может быстро определить:**
- какие цифровые активы существуют;
- какие связи между ними присутствуют;
- какие риски являются наиболее критичными;
- каким объектам можно доверять;
- какие последствия приведет компрометация конкретного объекта.

## Решение

DTEK Core создает цифровую модель организации. Каждый объект получает цифровой паспорт доверия. Все объекты объединяются в граф доверия. На основе анализа объектов, связей, угроз и рисков рассчитывается уровень доверия к каждому объекту организации.

## Целевая аудитория

- специалисты по информационной безопасности;
- руководители подразделений ИБ;
- системные администраторы;
- владельцы бизнеса;
- государственные организации;
- образовательные учреждения;
- промышленные предприятия.

## Стратегическая цель

Создать для управления цифровым доверием и информационной безопасностью платформу, DTEK Core должен стать стандартом управления цифровым доверием организаций в России.
