| Сложная задача № 3                                                                                                                                                                         |
| **Условие:**<br><br>Создайте конструкцию, которая будет включать в себя оператор **ИначеЕсли** и **Иначе**.                                                                                |
| **Задача:**<br><br>В условиях **Если**  и **ИначеЕсли** укажите ложные условия. Так как ни одно условие не сработает, пусть программа выводит Вам сообщение, где будет указан Ваш возраст. |
### ***Решение*** 

```
Сообщение = Новый СообщениеПользователю;
Если МойВозраст = 20 Тогда
	Сообщение.Текст = "Вам 20 лет";
	Сообщение.Сообщить(); 
ИначеЕсли МойВозраст = 35 Тогда
	Сообщение.Текст = "Вам 35 лет";
	Сообщение.Сообщить();
Иначе  
	Сообщение.Текст = "Вам " + МойВозраст + " лет"; 
	Сообщение.Сообщить();
КонецЕсли;
```

| Сложная задача № 4                                                                                                                              |
| **Условие:**<br><br>Создайте конструкцию, которая будет включать в себя три оператора **ИначеЕсли**.                                            |
| **Задача:**<br><br>Условия придумайте самостоятельно, попробуйте поигратьсяи включить фантазию, главное чтобы вы поняли как работает программа. |
### ***Решение***

```
Сообщение = Новый СообщениеПользователю;

Если МойВозраст > 16 Тогда 
	Сообщение.Текст = "Вы допущены до соревнований";
	Сообщение.Сообщить();
ИначеЕсли МойВозраст < 16 Тогда 
	Сообщение.Текст = "Вы не допущены до соревнований";
	Сообщение.Сообщить();
ИначеЕсли МойВозраст <= 18 Тогда 
	Сообщение.Текст = "Ваша катеогрия Юниоры";
	Сообщение.Сообщить();
ИначеЕсли МойВозраст >= 18 Тогда 
	Сообщение.Текст = "Ваша категория Взрослые";
	Сообщение.Сообщить();
Иначе 
	Сообщение.Текст = "Соревнования закончились";
	Сообщение.Сообщить();
КонецЕсли
```


| Задача № 6                                                                                                                                                   |
| **Условие:**<br><br>```1c<br>Если ЭтоОптоваяяПродажа Тогда<br>    КоэффициентНаценки = 1.10;<br>Иначе<br>    КоэффициентНаценки = 1.25;<br>КонецЕсли;<br>``` |

### ***Решение***

```
ОптоваяПродажа = Истина;

Сообщение = Новый СообщениеПользователю; 
Сообщение.Текст = РезультатКоофицентаНаценки;
Сообщение.Сообщить();
```
