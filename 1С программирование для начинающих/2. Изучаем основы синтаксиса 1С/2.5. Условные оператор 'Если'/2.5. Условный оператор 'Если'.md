### ***Оператор Если***

***Данный оператор управляет выполнением кода, основываясь на логических выражениях:*** 
```
	Если (Условие1) Тогда 
		// Блок кода, когда Условие1 выполняется
		// ...
	КонецЕсли;
```

***Пример кода:*** 
```
	МоеИмя = "Василий";
	
	Если МоеИмя = "Василий" Тогда 
		Сообщение = Новый СообщениеПользователю;
		Сообщение.Текст = "Меня зовут Василий";
		Сообщение.Сообщить();
	КонецЕсли;
```

В конструкции между словами **`Если`** и **`Тогда`** мы написали определенное условие, и если значение этого условия = **ИСТИНА**, то написанный код, который мы написали между словами **`Тогда`** и **`КонецЕсли` будет выполняться.**

Если значение условия будет **ЛОЖЬ**, тогда код написанный между словами **`Тогда`** и **`КонецЕсли` выполняться не будет!**

***Первый оператор `Иначе` работает следующим образом:***
```
	МоеИмя = "Василий";
	
	Сообщение = Новый СообщениеПользователю;
	Если МоеИмя = "Дмитрий" Тогда 
		Сообщение.Текст = "Ваше имя - Дмитрий";
		Сообщение.Сообщить();
	Иначе
		Сообщение.Текст = "Я не знаю Ваше имя :(";
		Сообщение.Сообщить();
	КонецЕсли;
```

То есть, если попытаться объяснить код своими словами, сначала программа проверит, равно ли **Истине** условие между **`Если`**  и **`Тогда`** _(МоеИмя = «Дмитрий»)_, в нашем случае условие равно **Ложь**, поэтому программа перейдет к оператору **`Иначе`** и выполнит код заложенный между **`Иначе`** и `**КонецЕсли**`

***Конструкция работы:*** 
```
	Если (Условие) Тогда 
		// Выполняется код, в случае если Условие = Истина
	Иначе
		// Выполняется код, в случае если Условие = Ложь
	КонецЕсли;
```

***Второй оператор `ИначеЕсли` работает следующим образом:***
```
	МоеИмя = "Василий";
	
	Сообщение = Новый СообщениеПользователю;
	Если МоеИмя = "Дмитрий" Тогда 
		Сообщение.Текст = "Ваше имя - Дмитрий";
		Сообщение.Сообщить();
	ИначеЕсли МоеИмя = "Василий" Тогда 
		Сообщение.Текст = "Ваше имя - Василий";
		Сообщение.Сообщить();
	Иначе
		Сообщение.Текст = "Я не знаю Ваше имя :(";
		Сообщение.Сообщить();
	КонецЕсли;
```

Главное различие этих двух операторов, что **`Иначе`** выполняется в случае, если ни одно из условий выше не выполнилось, а **`ИначеЕсли`** выполняется только в случае, если его условие выполнилось и условия операторов выше не выполнялись

***Логика работы:*** 
```
	Если Условие1 Тогда 
		// Выполняется код, в случае, когда Условие1 = Истина
	ИначеЕсли Условие2 Тогда 
		// Выполняется код, в случае, когда Условие2 = Истина
	Иначе
		// Выполняется код, в случае, когда не выполняется ни одно из условий
	КонецЕсли;
```

***Пример кода:*** 
```
	МоеИмя = "Василий";
	
	Сообщение = Новый СообщениеПользователю;
	Если МоеИмя = "Василий" Тогда 
		Сообщение.Текст = "Ваше имя - Василий";
		Сообщение.Сообщить();
	ИначеЕсли МоеИмя = "Василий" Тогда 
		Сообщение.Текст = "Ваше имя - Дмитрий";
		Сообщение.Сообщить();
	Иначе
		Сообщение.Текст = "Я не знаю Ваше имя :(";
		Сообщение.Сообщить();
	КонецЕсли;
```

Как мы можем увидеть, условия у операторов **`Если`** и **`ИначеЕсли`** одинаковые, исходя из этого можно подумать, что код, который в них прописан должен выполниться, **но это не так!**

Программа увидит, что первое условие равно **Истина**, и выполнит только код заложенный между **`Тогда`** и `**ИначеЕсли**`, дальше программа перестанет читать данную конструкцию и перейдет в конец конструкции в `**КонецЕсли**`.

# ***Тернарный оператор в 1С***

***Конструкция:*** 
```
?(<Логическое выражение>, <Выражение 1>, <Выражение 2>)
```

***Тернарный оператор в языке 1С*** - позволяет вычислить одно из двух заданных выражений в зависимости от результата вычисления логического выражения 

![[Pasted image 20260616101856.png]]

- В **первом** параметре мы передаем **условие**
- Во **втором** (через запятую) - результат, который будет если условие равно Истине
- В третьем - результат, который будет, если условие равно Ложь

# ***Операторы И ИЛИ НЕ и Операции сравнения (=, >, <, <>)***

*Операторы **`И` `ИЛИ` `НЕ`** используются в 1С для создания сложных комбинированных условий.*
## ***Оператор И***

```
МоеИмя = "Василий";
МойВозраст = 31;
```

***Можно проверить данные переменные без оператора `И`:*** 
```
Если МоеИмя = "Василий" Тогда
	Если МойВозраст = 31 Тогда
		Сообщение = Новый СообщениеПользователю;
		Сообщение.Текст = "Меня зовут Василий и мне 31 год";
		Сообщение.Сообщить();
	КонецЕсли;
КонецЕсли;
```

***Теперь решим данную задачу с оператором `И`:*** 
```
Если МоеИмя = "Василий" И МойВозраст = 31 Тогда
	Сообщение = Новый СообщениеПользователю;
	Сообщение.Текст = "";
	Сообщение.Сообщить();
КонецЕсли;
```
## ***Оператор ИЛИ***

```
СуммаДокумента = 3990;
ДокументПроведен = Ложь;
```

***Проверим при помощи ИЛИ верно ли хоть одно условие:*** 
```
Если СуммаДокумента = 0 ИЛИ ДокументПроведен = Ложь Тогда
	Сообщение = Новый СообщениеПользователю;
	Сообщение.Текст = "Сумма документа равна нуль или он не проведен";
	Сообщение.Сообщить();
КонецЕсли;
```

Как видим, одно условие верное: **`ДокументПроведен = Ложь`**, а другое нет: **`СуммаДокумента = 0`**. Но, так как у нас два условия соединены оператором **`ИЛИ`**, то код заложенный внутрь условного оператора `Если` - **выполнится**
## ***Оператор НЕ***

***Оператор `НЕ`*** - переворачивает значение любого условия 

```
ЭтоКонтрагент = Ложь;
```

```
Если НЕ ЭтоКонтрагент Тогда
	Сообщение = Новый СообщениеПользователю;
	Сообщение.Текст = "Это поставщик";
	Сообщение.Сообщить();
КонецЕсли;
```

***Решения без оператора `НЕ`:*** 
```
Если ЭтоКонтрагент = Ложь Тогда
	Сообщение = Новый СообщениеПользователю;
	Сообщение.Текст = "Это поставщик";
	Сообщение.Сообщить();  
КонецЕсли;
```

```
Если ЭтоКонтрагент Тогда
Иначе
	Сообщение = Новый СообщениеПользователю;
	Сообщение.Текст = "Это поставщик";
	Сообщение.Сообщить();      
КонецЕсли;
```
## ***Операции сравнения (=, >, <, <>)***

```
ДнейОтсрочкиПоДоговору = 30;
ДнейПросрочки = 44;
```

```
КолДнейПросрочки =  ДнейПросрочки - ДнейОтсрочкиПоДоговору;

Сообщение = Новый СообщениеПользователю;
Если КолДнейПросрочки > 0 Тогда
	Сообщение.Текст = "Контрагент имеет просроченную задолженность уже " +  КолДнейПросрочки + " дней.";
	Сообщение.Сообщить();
ИначеЕсли КолДнейПросрочки < 0 Тогда
	
	Сообщение.Текст = "Задолженность контрагента просрочится через: " + (-КолДнейПросрочки) + " дней.";
	Сообщение.Сообщить();
Иначе
	Сообщение.Текст = "Завтра задолженность контрагента просрочится!";
	Сообщение.Сообщить();
КонецЕсли;
```

`<>` - оператор не равно 

```
СтатусДокумента = "Согласован";
```

```
Если СтатусДокумента <> "Согласован" Тогда
	Сообщение = Новый СообщениеПользователю;
	Сообщение.Текст = "Документ еще не прошел процедуру согласования!";
	Сообщение.Сообщить();
КонецЕсли;
```
