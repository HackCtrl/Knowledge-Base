## **Даты**

***Для выполнение операции сложение двух дат используется специфичная конструкция. Чтобы сложить две даты, необходимо вычесть из второй даты "Пустую дату":*** 

- `Дата1 + (Дата2 - ПустаяДата)`

```
Сообщение = Новый СообщениеПользователю; 
Сообщение.Текст = '19900303' + ('19951224' - Дата(1, 1, 1)); 
Сообщение.Текст = Дата(1990, 3, 3) + (Дата(1995, 12, 24) - Дата(1, 1, 1)); 
Сообщение.Сообщить();
```

В 1С при сложении двух дат (например, если вы хотите добавить к дате какое-то количество дней, месяцев или лет) иногда используется операция **вычитания пустой даты**. Это необходимо для того, чтобы корректно преобразовать вторую дату в **интервал времени**, который будет добавлен к первой дате

**`Дата(1, 1, 1)`** — это специальная "нулевая" дата в 1С, которая используется как отправная точка для вычисления разницы между датами

| Значение | Что это в 1С |
| -------- | ------------ |
| 0 дней   | 01.01.0001   |
| 1 день   | 02.01.0001   |
| 2 дня    | 03.01.0001   |
| 3 дня    | 04.01.0001   |
| 4 дня    | 05.01.0001   |
| 5 дней   | 06.01.0001   |
***Например, чтобы вычесть из нашей даты 7 дней необходимо написать следующий программный код:*** 

```
ДатаРождения = '19940610';
НеделяДоДатыРождения = '19940610' - 24*60*60*7; 
Сообщение = Новый СообщениеПользователю;
Сообщение.Текст = НеделяДоДатыРождения;
Сообщение.Сообщить(); 
```
## **Булево**

В 1С тип булево в выражениях представляется числовыми значениями: "Истина" как 1, а "Ложь" как 0. Поэтому при сложении булевых значений происходит автоматическое преобразование

```
Истина + Истина будет равно 2  
Истина + Ложь будет равно 1  
Ложь + Ложь будет равно 0
```

```
Сообщение = Новый СообщениеПользователаю; 
Сообщение.Текст = Истина + Лож; 
Сообщение.Сообщить(); 
```
# ***Разность/Вычитание***

## **Строки**

  В 1С программировании операция вычитания для строк вызовет ошибку, строковые операции обычно ограничиваются конкатенацией (объединением) строк с помощью оператора "+" или использованием специальных функций для работы со строками. Если вы пытаетесь выполнить арифметическую операцию с двумя строками, как если бы они были числами, это вызовет ошибку, поскольку операции вычитания не определены для строк в языке 1С
## **Числа**

```
Сообщение = Новый СообщениеПользователю; 
Сообщение.Текст = 12 - 5 - 1;
Сообщение.Сообщить(); 
```
## **Даты**

***Для операции вычитания одной даты из другой, необходимо по аналогии со сложением из второй даты вычесть пустую дату:***

```
Сообщение = Новый СообщениеПользователю; 
Сообщение.Текст = Дата(1995, 12, 24) - (Дата(1990, 1, 1) - Дата(1, 1, 1))
Сообщение.Сообщить();
```
## **Булево**

```
Сообщение = Новый СообщениеПользователю;
Сообщение.Текст = Истина - Лож; 
Сообщение.Сообщить();
```
# ***Применение `Формат()` для удобного представления значений***

```
Сообщение = Новый СообщениеПользователю;
Сообщение.Текст = ИзмененноеРешение;
Сообщение.Сообщить();
```
# ***Примеры работы с функцией `Формат()`***

***Примеры форматирования чисел***
```
Пример = Формат(123456.789, "ЧЦ=10; ЧДЦ=2"); // А = "123 456,79"
Пример = Формат(123456.789, "ЧРД='-'"); // А = "123 456-789"
Пример = Формат(-123456.789, "ЧО=0"); // А = "(123 456,789)"
Пример = Формат(123456.789, "ЧЦ=10; ЧДЦ=2; ЧФ=""Ч руб."""); // А = "123 456,79 руб."
Пример = Формат(100, "ЧФ=""$Ч' / Час'"""); // А = "$100 / Час"
```

***Примеры форматирования дат***
```
Пример = Формат('20020820153309', "ДФ=""дд ММММ гггг 'г.' ЧЧ:мм:сс"""); // А = "20 августа 2002 г. 15:33:09"
Пример = Формат('20020820153309', "ДФ=""дд/ММ-гггг"""); // А = "20/08-2002"
Пример = Формат('20020820153309', "ДЛФ=ДД"); // А = "20 августа 2002 г."
Пример = Формат('20020820153309', "ДЛФ=Д"); // А = "20.08.2002"
Пример = Формат('20020820153309', "ДЛФ=В"); // А = "15:33:09"
```

***Примеры форматирования логических выражений***
```
Пример = Формат(Истина, "БЛ=Отсутствует; БИ=Доступен"); // А = "Доступен"
```
