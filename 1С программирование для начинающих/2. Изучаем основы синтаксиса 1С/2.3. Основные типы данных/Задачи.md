### ***Решение***

*1 способ решения*
```
МойЛюбимыйЦвет = "Синий";   
МоеЛюбимоеЧисло = 8;    
МояДатаРождения = '20070608';   
МойПол = "Мужчина"; 

Сообщение = Новый СообщениеПользователю;  
Сообщение.Текст = МойЛюбимыйЦвет;   
Сообщение.Сообщить();

Сообщение.Текст = МоеЛюбимоеЧисло;   
Сообщение.Сообщить();

Сообщение.Текст = Формат(МояДатаРождения, "ДФ=yyyy-MM-dd");      
Сообщение.Сообщить();

Сообщение.Текст = МойПол;   
Сообщение.Сообщить();
```

Так же можно сделать объяединением, то есть использовать оператор `+`, чтоб объеденить наши переменные в одну строку и при помощи функции `Символы.ПС`, сделать так, чтоб они переходили на новую строку
