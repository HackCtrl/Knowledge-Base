***Цикл*** - это управляющая конструкция, которая позволяет нам выполнять программный код нужное количество раз 

***Синтаксис простейшего цикла:***  
```
	Пока Условие Цикл
		// Код, который будет выполняться в цикле, пока срабатывает условие
	КонецЦикла;
```

*В данном примере мы видим, что между **Пока** и **Цикл** мы указываем какое-то условие, и если это условие равно **Истина** то код, который указан между **Цикл** и **КонецЦикла** будет выполняться пока условие не будет равно **Ложь***

***Пример цикла:*** 
```
ПроверяемоеЧисло = 1;
	Сообщение = Новый СообщениеПользователю;

	Пока ПроверяемоеЧисло < 10 Цикл 
		Сообщение.Текст = ПроверяемоеЧисло;
		Сообщение.Сообщить();
		ПроверяемоеЧисло = ПроверяемоеЧисло + 1;
	КонецЦикла;
```
