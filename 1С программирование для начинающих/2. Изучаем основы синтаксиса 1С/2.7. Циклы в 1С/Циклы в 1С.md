***Цикл*** - это управляющая конструкция, которая позволяет нам выполнять программный код нужное количество раз 

***Синтаксис простейшего цикла:***  
```
	Пока Условие Цикл
		// Код, который будет выполняться в цикле, пока срабатывает условие
	КонецЦикла;
```

*В данном примере мы видим, что между **Пока** и **Цикл** мы указываем какое-то условие, и если это условие равно **Истина** то код, который указан между **Цикл** и **КонецЦикла** будет выполняться пока условие не будет равно **Ложь***

***Пример цикла:*** 
```
ПроверяемоеЧисло = 1;
Сообщение = Новый СообщениеПользователю;

Пока ПроверяемоеЧисло < 10 Цикл 
	Сообщение.Текст = ПроверяемоеЧисло;
	Сообщение.Сообщить();
	ПроверяемоеЧисло = ПроверяемоеЧисло + 1;
КонецЦикла;
```

***Следующий вид цикла***

```
	Для Счетчик = НачальноеЗначение По КонечноеЗначение Цикл 
		// Код, который будет выполняться в цикле, пока Счетчик не достигнет КонечноеЗначение
	КонецЦикла;
```

Например, нам нужно вывести слово «Привет» пользователю 7 раз. Мы можем использовать предыдущий вид цикла **Пока** и считать переменную в цикле, но проще это сделать используя конструкция **Для**

***Пример решения с помощью цикла **Для*****

```
	Сообщение = Новый СообщениеПользователю;
	Для КоличествоСообщений = 1 По 7 Цикл
		Сообщение.Текст = "Привет!";
		Сообщение.Сообщить();
	КонецЦикла;
```

***Реализация с помощью цикла Пока***

```
	ПроверяемоеЧисло = 1;
	Сообщение = Новый СообщениеПользователю;

	Пока ПроверяемоеЧисло <= 7 Цикл 
		Сообщение.Текст = "Привет!";
		Сообщение.Сообщить();
		ПроверяемоеЧисло = ПроверяемоеЧисло + 1;
	КонецЦикла;
```

***Конструкция для цикла Для***
- `Для По Цикл КонецЦикла`
