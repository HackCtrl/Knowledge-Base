***Информационная база*** - это экземпляр одного прикладного решения 1С-Предприятие. Она представляет собой логически целостную систему, включающую две конфигурации (как минимум), базу данных, а так же дополнительную информацию, необходимую для администрирования 

***Конфигурация*** - это программа, которую мы будем создавать в нашей ИБ

***Конфигурация*** - это программа, которая исполняется 
***База данных*** - это данные, которые вводят и изменяют пользователи

# ***Файлы CF  и DT***

# ***DT-файл***

Файл формата `.dt` нужен, чтобы выгружать ИБ. То есть все данные, которые мы завели в режиме 1С Предприятие, так же будут сохранены в данный файл

***DT-файл*** - это выгрузка ИБ, копия всей базы, как конфигурации, так и всех данных 

***Создания файла копии:*** 

`**Администрирование» => «Выгрузить информационную базу**`

***Загрузка ИБ:*** 

`**Администрирование" => "Загрузить информационную базу**`

**Важно!** *Перезапишутся все изменения, которые мы вносили* 
# ***CF - файл***

***Сохранение конфигурации:*** 

`**Конфигурация» => «Сохранить конфигурацию в файл**`

***Загрузка конфигурации:*** 

`**Конфигурация" => "Загрузить конфигурацию из файла**`
