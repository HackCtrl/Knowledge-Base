### ***Cross-Site Scripting (XSS)***

Это атака на клиент-сайт пользователя 

- ***Reflected XSS*** - позволяет выполнить HTML и JS на стороне браузера клиента 
- ***Stored XSS*** - приложение позволяет хранить данные пользователя 
- ***DOM XSS*** - модификация Document Object Model с целью запуска JS
