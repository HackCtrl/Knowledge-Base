***Способы получения привилегий:*** 
- ***Разведка*** 
- ***Lsass*** - критически важный системный процесс в ОС Windows, который отвечает за управление политиками безопасности, аутентификацию пользователей, хранение и обработку учетных данных 
- ***SAM*** - диспетчер учетных записей 
- ***MITM*** - тип кибератаки, при которой злоумышленник незаметно перехватывает и при необходимости изменяет коммуникацию между двумя сторонами 
- ***NBNS и LLMNR*** - (**LLMNR**) - протокол, который работает на уровне локальной сети и заменяет DNS, когда DNS сервер не доступен. (**NBT-NS**) - протокол, который отвечает за регистрацию и разрешение имен в рамках NetBIOS
- ***SMB Relay*** - тип кибератаки, которая использует уязвимости протокола SMB, широко применяется для сетевого обмена файлами в средах Windows
- ***GPP*** - это стандартизированный фреймворк для унификации управления согласием пользователей и соблюдением норм конфиденциальности в цифровой среде
- ***Golden ticket*** - тип атаки, при которой злоумышленник получает несанкционированный и длительный доступ к сети за счет подделки аутентифицированных билетов в системе Active Directory
