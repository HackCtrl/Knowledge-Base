# ***Мотивация*** 

1. Конкурентная борьба 
- **Области интересов** 
	- Ослабление конкурентов и увеличение репутационных потерь организации 
	- Фальсификация документов 
	- Хищение образцов интеллектуальной собственности 
	- Саботаж 

***Файл `robot.txt***`*** - это файл который хранит настройки для индексации страниц в интернете 

2. Манипуляция финансовой информацией 
- ***Область интересов*** 
	- Платежные шлюзы 
	- Системы межбанковского обмена и АБС (автоматизированные банковские системы)
	- Мошеннические действия при оплате товаров и услуг через интернет 
	- Банк-клиенты 
	- Мошенничество с банковскими картами 

3. Вымогательство и шантаж 
- ***Область интересов*** 
	- Любая критичная для бизнеса/физического лица информация 
	- Нарушение нормальной деятельности организации 
	- Компрометация ключевых персон бизнеса 
	- Отказ в обслуживании
	- Репутационные риски 

4. Перепродажа данных 
- ***Область интересов***
	- Любая критичная для бизнеса или физического лица информации 
	- Карточные данные 
	- Персональные данные 
	- БД государственного сектора 

5. Вычислительные ресурсы
- ***Область интересов*** 
	- Вычислительные ресурсы (майнинг, расчет, генома, лекарств)
	- Криптоджекинг (скрытый майнинг)
	- Организация анонимного доступа в сеть (proxy)
	- Доступ в инфраструктуру с последующей перепродажей

6. Месть 
- ***Область интересов*** 
	- Увеличение репутационных и финансовых потерь организации 
	- Нарушение нормальной деятельности организации 
	- Отказ в обслуживании 
	- Саботаж 
	- Порча данных 
	- АСУТП
