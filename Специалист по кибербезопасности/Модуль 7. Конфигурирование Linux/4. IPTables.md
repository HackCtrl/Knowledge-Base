Это утилита командной строки для управления встроенным в ядро Linux межсетевым экраном (брандмауэром) на базе проекта Netfilter. Она позволяет настраивать фильтрацию сетевого трафика, преобразование сетевых адресов, модификацию пакетов и другие операции. Проверяет входящие и исходящие сетевые пакеты на соответствие заданным правилам и применяет к ним указанные действия (например, разрешение или блокировку трафика

**Таблицы.** Правила организованы в таблицы, каждая из которых предназначена для выполнения определённых задач. Всего существует несколько таблиц, например:
- `filter` — основная таблица для фильтрации пакетов, используется по умолчанию
- `nat` — отвечает за преобразование сетевых адресов (NAT), например, проброс портов
- `mangle` — используется для модификации заголовков пакетов (изменение TTL, TOS и других параметров)
- `raw` — обрабатывает пакеты до определения состояний соединений, используется для исключения пакетов из отслеживания
- `security` — применяется для мандатного управления доступом (например, в SELinux)
