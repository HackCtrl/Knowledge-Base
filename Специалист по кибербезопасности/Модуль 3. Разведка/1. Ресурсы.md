***VirusTotal*** - можно загрузить любой файл и проверить его сигнатуры. В нем около 60 антивирусов, которые дают нам результат 

***nmap*** - утилита для диагностики сети (проверяют доступные порты, открытые порты и так далее)
- `-T` - опция для тех, кто хочет остаться незаметным (`0-5`) 5 вариантов сканирования `0` - между запросами большой промежуток времени для того, чтобы нас не обнаружили (может сканировать большой диапазон IP только в течении суток). `1` - чуть быстрее, но тоже достаточно медленно. `2` - нормальный и так далее. Обычно используют -`T4` (обычный режим использования если мы хотим остаться не заметным)
- `--skript=banner` - рисует баннер сервиса, который отвечает с той стороны 
- `-sU` - для сканирования UDP
- `--osscan-guess` - для того, чтобы узнать ОС
