***VipNet*** - это средство криптографической защиты информации. Основным требованием законодательства, использование СКЗИ - **передача конфиденциальной информации по каналам связи, не защищенным от перехвата нарушителем передаваемой по ним информации или от несанкционированного доступа воздействия на эту информацию** 

***Контролируемый канал связи*** - не обязательно использования СКЗИ 
***Не контролируемый*** - обязательно используем СКЗИ

# ***Основные элементы сети VipNET*** 

![[Pasted image 20260214110454.png]]

***Сетевой узел*** - это защищенная машина 

***VipNET Client:*** 

1. ***Организуется шифрование трафика на ПК*** 
- Возможность удаленного расположения клиента 
2. ***Обязательно регистрируется на VipNET Coordinator***
- Организация защищенного канала - через сервер соединение (Координатор)
- Получает информацию о сети - через сервер IP-адресов (Координатор)
3. ***Наличие деловой почты*** 
- Обмен письмами и файлами через отдельное приложение 

***VipNET Coordinator:*** 

1. ***Координатор - сетевое оборудование*** 
- Маршрутизация трафика, функции МЭ, настройка сервисов 
1. ***За координатором ремитируются клиенты*** 
- Координаторы обмениваются служебной информацией 
- Координаторы выполняют роли сервера соединений и IP-адресов 
1. ***Существуют различные версии VipNET Coordinator*** 
- ПО: Windows / Linux 
- ПАК: HW50 / HW100 / HW1000 / HW2000 / HW5000
1. ***Туннелируют незащищенные машины*** 
- Подключают к защищенной сети машины без ПО VipNET Client 
- Выполняет функции шифрования трафика от незащищенных машин
