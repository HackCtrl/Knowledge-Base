*Фрилансер закрывает проектную занятость* 

***Фрилансер*** - это человек, который выполняет определенный тип работ, как проект (не каждый человек, который делает проект - фрилансер, за то каждый фрилансер выполняет проектную работу). Это человек, который работает над проектами на постоянной основе 
### ***Три вида фриланса*** 

1. ***Чистый фриланс*** - фрилансер, который работает только на себя, без привязки к какой либо организации 

***Плюсы:*** 
- Свобода выбора проекта 
- Сроки 
- Бюджеты 
- Дедлайны 

***Минусы:*** 
- Высокий риск непостоянного дохода 

*Подойдет для тех, кто развивает личный бренд и высокую экспертизу у кого есть проработанный канал с коммуникацией, с потенциальными клиентами, хороший имидж, который приведет клиентов по сарафанному радио* 

2. ***Работа + фрилансер*** - это специалист, который работает на стандартной схеме найма, дополнительно получает доход с заказов

***Плюсы:*** 
- Есть стабильный приход дохода 
- Можно выбирать проекты, которые хочется 

***Минусы:*** 
- Риск конфликтов с основным работодателем 
- Риск выгорания 

*Для тех кто хочет поднять уровень заработка* 

3. ***Фриланс + частичная занятость*** - который помимо рядовых работ, работает на частичной занятости или на каком то долгосрочном проекте 

***Плюсы:*** 
- Идеальный баланс стабильности и выбора проектов 

***Минусы:*** 
- Иногда работа по частичной занятости недостаточно амбициозная
