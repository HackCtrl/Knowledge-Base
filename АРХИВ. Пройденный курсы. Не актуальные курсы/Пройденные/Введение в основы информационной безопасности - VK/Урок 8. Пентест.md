# ***Кто такой пентестер ?*** 

***Пентестер*** - специалист по тестированию на проникновение 
***Белый хакер*** - это инженер, который специализируется на аудите безопасности ИС компании, независимо от ее стека технологий 

*Задачей специалиста по тестированию на проникновение является поиск уязвимостей ИС компании* 
# ***Роль пентестера в ИБ*** 

- Взлом ИС 
- Создание архитектуры ИС 
- Внутри ИС 
# ***Виды пентестов*** 

- Внутренний пентест 
- Внешний пентест 
- Облачный пентест (облачные провайдеры)
- K8s + CI/CD 
- Финансовый сектор 
- Смарт-контракты 
- Мобильные приложения 
- Черный, белый и серый ящик ящики 
# ***Основные методики*** 

- Ручное тестирование 
- Автоматизация (сканеры)
- Работа с SOC 
- Обучение технических сотрудников 
- Работа с Bug Bounty (проверки критов)
# ***Примерный ход тестирования***

- Разведка 
- Сканирование 
- Анализ уязвимостей 
- Эксплуатация 
- Отчетность 
- Повторение тестирования 
# ***Взаимодействие с SOC (Purple Team)***

- Проверить анализ разработанных алертов 
- Мониторить тренды и пушить новые проверки 
- Записывать флоу атак по времени для легкого разбора атак 
# ***Взаимодействие с Bug Bounty*** 

- Если в компанию прилетает критическая уязвимость (крит), пентестер может копнуть глубже 
- Компрометация до конечной уязвимой точки 
- Поиск IOC
