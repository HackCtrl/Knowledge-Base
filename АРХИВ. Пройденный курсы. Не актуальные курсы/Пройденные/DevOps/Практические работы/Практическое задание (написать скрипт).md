Мы с вами писали на Bash скрипт, который спрашивал, как нас зовут, из какого мы города и хотим ли продолжить опрос, а также выводил всю информацию на экран. Скрипт был цикличным, и, пока мы отвечали на вопрос о продолжении опроса положительно: «yes», — скрипт повторял свою работу. Как только мы отвечали «no», скрипт заканчивал работу и прощался. Его необходимо переписать на синтаксис PowerShell. Аналогом `read` в Bash для PowerShell является `Read-Host` и имеет конструкцию `$var = Read-Host "Please write something"`. Удачи!

```
$again = "yes"
echo $again
while ($again -eq "yes") {
	$Name = Read-Host "Пожалуйста, введите имя"
	echo "Вас зовут: $Name"
	
	$City = Read-Host "Из какого вы города"
	echo "Ваш город: $City"
	
	$again = Read-Host "Повторим"
}
echo "До свидания"
```
