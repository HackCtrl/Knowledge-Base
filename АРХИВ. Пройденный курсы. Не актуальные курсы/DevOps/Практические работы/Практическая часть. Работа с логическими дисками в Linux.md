### **ШАГ 7: СОЗДАНИЕ ФАЙЛОВОЙ СИСТЕМЫ И МОНТИРОВАНИЕ**

***7.1. Создадим ФС ext4 на логическом томе***
- `sudo mkfs.ext4 /dev/data/lv`
![[Pasted image 20260102143331.png]]

***7.2. Создадим точку монтирования*** 
- `sudo mkdir /mnt/data`

***7.3. Монтируем логический том*** 
- `sudo mount /dev/data/lv /mnt/data`

***7.4. Проверим монтирование*** 
- `df -hT /mnt/data`
![[Pasted image 20260102143545.png]]

### **ШАГ 9: ФИНАЛЬНАЯ ПРОВЕРКА ВСЕЙ СТРУКТУРЫ**

***9.1. Запустим полную проверку*** 
```
echo "=== 1. СТРУКТУРА ДИСКОВ (lsblk) ==="
lsblk

echo -e "\n=== 2. LVM СТРУКТУРА ==="
echo "Физические тома (PV):"
sudo pvs
echo -e "\nГруппы томов (VG):"
sudo vgs
echo -e "\nЛогические тома (LV):"
sudo lvs

echo -e "\n=== 3. ФАЙЛОВЫЕ СИСТЕМЫ ==="
df -hT | grep -E "(Filesystem|data)"

echo -e "\n=== 4. ПУТИ К УСТРОЙСТВАМ LVM ==="
ls -la /dev/mapper/ | grep data
ls -la /dev/data/

echo -e "\n=== 5. СОДЕРЖИМОЕ /mnt/data ==="
ls -la /mnt/data/
```

***9.2. Проверим содержимое файлов*** 
```
echo -e "\n=== 6. СОДЕРЖИМОЕ ФАЙЛОВ ==="
sudo cat /mnt/data/practic.txt
sudo cat /mnt/data/snapshot_test.txt
sudo cat /mnt/data/test_dir/file_1.txt
```
