***В Python разные типы чисел:***
- `int` - целые числа 
- `float` - числа с плавающей точкой 
- `complex` - комплексное число
## ***Тип int***

К типу `int` относятся все целые и отрицательные числа без дробной части (0 - это тоже `int`)

Для лучшей читаемости целые числа в Python можно записывать, разделяя разряды нижним подчеркиванием `432246123 ⇒ 432_246_123`

***Переменные типа `int` , как и переменны других типов, не требует отдельного объявления:*** Python автоматически присвоит переменным нужный тип при операции присваивания

```
x = 231 # это int 
y = 0 # int 
z = -100 # int
```
## ***Арифметические операции***

***Арифметические операции*** - это выражение, состоящий из операндов и арифметических операторов

![[Pasted image 20260411120626.png]]

***Операнд*** - это значение над которыми проводится операция

***Арифметический оператор в Python*** - это символ, который выполняет операцию над несколькими операндами ( `+, -, *` )
## ***Применение арифметических операторов***

### ***Сложение и вычитание***

```
# согласно правилам оформления PEP8 операторы отделяются пробелами отсуствие пробелов не приведт к ошибке, но лучше соблюдать 

x = 40 -11 
print(x) 
# Вывод в терминал - 29 

x = 7 + 12 
print(x) 
# Вывод в терминал - 19
```

В фитнес треке встроен датчик шагов - каждый шаг добавляется в переменную `steps`, каждый шаг пользователя, добавляется единица в переменную `steps`

```
# к предыдущему значению steps прибавляем единицу и полученное значение перезаписываем в ту же самую переменную 

steps = steps + 1
```

Это называется инкрементирование (увеличение значения переменной), так же есть декрементирование (уменьшение значения переменной)

***В python инкрементирование можно записывать через комбинированный оператор присваивания:***

```
# Выражение 
steps = steps + 1 

# Комбинированный оператор присваивания 
steps += 1
```

***Комбинированные операторы присваивания всех операций:***
- `-= 
- `*=` 
- `/=`
## ***Умножение***

```
x = 5 * 2 
print(x) 
# Вывод в терминал - 10
```
## ***Возведение в степень***

Синтаксис возведения числа в степень - `число**степень`

```
x = 5**2 
print(x) 
# Вывод в терминал - 25
```
## ***Целочисленное деление***

***Операция целочисленного деления - это деление с остатком, в результате такой операции получается неполное частное и остаток. Оператор целочисленного деления возвращает неполное частное:***

```
x = 22 // 2 
print(x) 
# Вывод в терминал - 11 (остатка нет) 

y = 22 // 3 
print(x) # Вывод в терминал - 7 (это результат деления с остатком, возвращаеся только неполное частное) 

z = 2 // 3 
print(z) # Вывод в терминал - 0 (неполное частное 0, остаток 2, но он не возвращается)
```
## ***Получение остатка от деления: оператор «модулю»***

***Этот оператор возвращает остаток от целочисленного деления:***

```
x = 11 % 3 
print(x) 
# Вывод в терминал 2
```
## ***Деление***

```
x = 20 / 2 
print(x) 
# Вывод в терминал - 10.0 

x = 11 / 3 
print(x) 
# Вывод в терминал - 3.6666666666666665
```

*Числа полученные в результате деления, Python воспринимает не как целые, даже если результат выглядит как целое число*

`type()` - тип передаваемого аргумента

```
two = 10 / 5 
print(type(two))
```
## ***Тип float***

*Этот тип применяется для представления вещественных чисел*

```
a = 2.345 
b = 2. 
c = 2.0 
x = .07 
y = 0.07
```

***Чтобы не потерять точность при вычислениях используют такой прием:*** перед обработкой число типа `float` домножают на 10 в степени n (на 10, 100, 1000...). Множитель подбирают так, чтобы дробная часть исходного числа стала равной нулю. Затем проводят необходимые операции, а после обработки полученное значение делят на то же число, на которое умножали, приводя разрядность числа в исходное состояние

### ***Модуль Decimal***

*Кроме встроенных типов данных в Python можно применять типы из подключенных модулей. Если точность при выполнении операций критична в Python есть встроенный модуль Decimal: он дает почти сто процентную точность при работе с дробями*

***Для объявления таких числе нужно:*** 
- Импортировать в код класс Decimal 
- При объявлении переменной указать этот класс
- Предать в конструктор класс значения числен в кавычках - в виде строки 

```
from decimal import Decimal
 
x = Decimal('3.3') 
print(type(x)) 

# Вывод в терминал: <class 'decimal.Decimal'>
```

***Сравнение типа `float` и `Decimal`:*** 
```
from decimal import Decimal 

i = 3.3 + 4.1 
print(i) 
# Вывод в терминал: 7.3999999999999995 

i = Decimal('3.3') + Decimal('4.1') 
print(i) 
# Вывод в терминал: 7.4
```
### ***Тип complex***

*Встроенный числовой тип в Python - комплексные числа. Они описываются двумя числами, одно из которых - мнимое* 
### ***Значение None***

***Зачастую при объявлении переменной на требуется присваивать ей значения. А иногда программа может передать в переменную пустое значение. Для этого в Python есть тип `None`:*** 

```
a = None 
print(type(a)) 
# Вывод в терминал: <class 'NoneType'> 

# Переопределим переменную: 
a = 1 
print(type(a)) 
# Вывод в терминал: <class 'int'>
```

*Часто применяется для указания значений по умолчанию в аргументах функций* 
### ***Приоритеты арифметических операций***

***Самый приоритетный арифметический оператор*** - возведение в степень, умножения, деления, получение остатка и целочисленное деление. ***Самые низкоприоритетные*** - операторы сложения и вычитания  

*При равном уровне приоритетов операторы выполняются по очереди, слева - направо* 

![[Pasted image 20260415110612.png]]

*Но если рядом будут находится две операции возведения в степень, то последовательность их выполнения будет обратной: справа - налево* 

![[Pasted image 20260415110723.png]]

***Чтобы изменить приоритетность выполнения операций*** - применяют скобки: операции в скобках выполняются в первую очередь 

```
print(1 + 2*3) 
# Вывод в терминал: 7 
# Но 
print((1 + 2) * 3) 
# Вывод в терминал: 9
```

Вместо пробелов между числами поставьте арифметические операторы и скобки так, чтобы в результате получилось наибольшее значение
