***Интерация*** - каждое повторение выполнение действия или ряда действия. Для работы с ними применяют циклы 
## ***Цикл for***

Он используется, когда есть блок кода, который нужно выполнить фиксированное количество раз. Этот цикл всегда используется в сочетании с итерируемым объектом, таким как список или диапазон 

![[Снимок экрана — 2026-05-21 в 09.33.34.png]]

***При объявлении цикла `for` указывают:*** 
- Переменную, в которой на каждой итерации будет передаваться очередной элемент последовательности 
- Имя последовательности, которую нужно перебрать ![[Снимок экрана — 2026-05-21 в 09.36.01.png]]

```
movie_ratings = [4.7, 5.0, 4.3, 3.1] 
# Переменная rating поочерёдно принимает элементы 
# последовательности movie_ratings. 
for rating in movie_ratings: 
	# Теперь переменную rating можно обработать в теле цикла. 
	if rating > 4.7: 
		print('Фильм крут')
```

Другой пример использования цикла for - перебор элементов строки и вывод их в терминал

## ***Итерация по диапазону чисел***

***Объект типа Range (с англ. «диапазон»)*** - это числовой диапазон, заданный по определенным правилам. Он итерируемый, и с ним можно работать так же, как и с коллекциями 

*Объект типа Range создаётся при вызове функции `range()`*

```
simple_range = range(1, 10) 
# Создали последовательность чисел от 1 до 9. 
print(type(simple_range)) 

# Вывод в терминал: <class 'range'>
```

***У функции `range()` может быть от одного до трёх аргументов, например:***
- `range(15)` вернет последовательность чисел от 1 до 14 включительно 
- `range (3, 15)`вернет последовательность чисел от 3 до 14 включительно 
- `range (3, 15, 2)` вернет последовательность чисел от 3 до 14 с шагом 2 (3, 5, 7, 9, 11, 13). А вот `range (15, 3, -2)` вернет (15, 13, 11, 9, 7, 5)

```
simple_range = range(1, 10, 2) 
print(simple_range) 

# Вывод в терминал: range(1, 10, 2) 
# А где числа-то?
```

В объекте типа Range хранятся не значения из диапазона, а лишь условия для вычисления этих значений. В результате объект Range занимает в памяти очень мало места, даже если хранит диапазон чисел от единицы до миллиарда.

***Чтобы из объекта range получить конкретное значение - нужно явно обратиться к нужному элементу:*** 

```
simple_range = range(1, 10, 2)
print(simple_range[3])

# Вывод в терминал: 7

- index 0 — число 1
- index 1 — число 3
- index 2 — число 5
- index 3 — число 7
- index 4 — число 9
```

***А вот и практическое применение `range()`: обработаем в цикле `for` диапазон и получим элементы с одинаковыми индексами из разных списков:***
```
movies = ['Матрица', 'Хакеры', 'Трон', 'Тихушники', 'Сеть'] movie_ratings = [4.7, 5.0, 4.3, 4.9, 3.4] 

# В качестве верхней границы диапазона 
# передаётся длина списка movies. 
print('Рейтинг пользователей') 
for index in range(len(movies)): 
print(movies[index] + ':', movie_ratings[index])
```
## ***Цикл while***

Это цикл используется, когда количество итераций неизвестно. Цикл будет выполнять вложенный в него код до тех пор, пока будет соблюдаться условие, заданное при его объявлении 

![[Снимок экрана — 2026-05-21 в 10.37.32.png]]

![[Снимок экрана — 2026-05-21 в 10.38.01.png]]

Например, хозяева киноклуба придумали простую игру: зрителям нужно угадать, какой фильм, из указанного списка, будет показан в очередной кинопоказ. Сам фильм выбирается программой случайным образом.

Сколько попыток понадобится пользователю чтобы угадать название? Неизвестно. Значит в дело вступает цикл `while`.

```
print('Какой фильм мы будем смотреть?') 
print(movies) 
answer = input() # Ожидаем ввод названия фильма от пользователя. 

while answer != movie: # Если не угадал, 
	answer = input('Попробуй еще: ') # то ждём ввода нового названия. 
	
print('Молодец, угадал!')
```

Если условие при объявлении цикла изначально ложно, то тело цикла не будет выполняться вообще.

```
x = 1 
while x > 10: 
	print('Этот текст никогда не будет выведен на печать!')
```

А если указать выражение, значение которого всегда будет истинным или вместо выражения просто написать `True`, то цикл будет выполняться бесконечно

```
while 1 == 1: 
	print('Эта строка будет выводиться бесконечно!')
```
