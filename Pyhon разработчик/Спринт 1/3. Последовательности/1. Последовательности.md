В программирование существует востребованная задача - хранить и обрабатывать наборы данных

***Например, нам нужно хранить число шагов в течении всей недели:***
```
monday_steps = 12000 
tuesday_steps = 9000 
wednesday_steps = 10000 
thursday_steps = 2500 
friday_steps = 14000 
saturday_steps = 8903 
sunday_steps = 9000
```

***Данный тип сложно обрабатывать, а если нам нужно будет за месяц или год, в таком случае лучше сохранить все в одну переменную с логичным названием:***
```
week_steps = [12000, 9000, 10000, 2500, 14000, 8903, 9000]
```

***Коллекции*** - наборы данных присвоенных одной переменной (сохраненные в один объект)
### ***Упорядоченные и неупорядоченные последовательности***

***Упорядоченные коллекции*** - к любому ее элементу можно обратиться по индексу (порядковому номеру элемента). Можно вывести все элементы по очереди и порядок будет один и тот же, такие коллекции называют последовательности

***Неупорядоченные коллекции*** - невозможно определить какой элемент первый, какой последний
### ***Виды последовательностей в Python***

***Несколько типов последовательностей (Sequence Type):***
- `list` - список 
- `tuple` - кортеж 
- `range` - диапазон

***Подклассы последовательностей:***
- `str` - строки
- Бинарные списки (Binary Sequence Types):
	- `bytes` - байты 
	- `bytearray` - массивы байт 
	- `memoryview` - специальные объекты для доступа к внутренним данным объекта через буфер обмена памяти

### ***Сравнение последовательностей***

При сравнении последовательностей поочередно сравниваются элементы с одинаковыми индексами, и при первом же различие определяется результат сравнения 

Сначала сравниваются элементы с индексом 0, и если они равны, сравниваются следующие два элемента и так пока не будет исчерпана любая последовательность

***Результатом сравнения будет будет логическое значение True или False :***
```
print([1,2,3] < [1,2,4]) 
# Вывод True
```

В данном случае первый два индекса (0 и 1) они равны, а вот индекс в первой последовательности (2) уже соответствует условию сравнения и поэтому результат True 

При сравнении строк принимаются во внимание символы и их регистр. Все текстовые символы (буквы, цифры, знаки препинания, пробел, иероглифы) имеют числовую кодировку. На основе этой кодировки и происходит арифметическое сравнение строк. Заглавные и строчные буквы - это разные символы, соответсвенно, и кодировка у них разная

***Группы символов можно распределить по возрастанию их числовых кодов:***
- Спецсимволы, знаки препинания и цифры: начинается с 32, это числовое значение символа 'пробел' 
- Латиница - заглавные буквы `A` = 65 
- Латиница - строчные буквы `a` = 97
- Расширенная латиница, диакритические знаки, греческое письмо 
- Кириллица - заглавные буквы `A` = 1040 
- Кириллица - строчные буквы `a` = 1072 И так далее

```
print('Слон' < 'слон') 
# Вывод в терминал True 
# Элемент с индексом 0 первой строки - это буквенный символ в верхнем регистре, его числовой код меньше той же буквы в нижнем регистре 

print ('1b' > 'fb') 
# Вывод в терминал False 
# Элемен с индексом 0 первой строки - цифра. ЕЕ числовой код меньшге чем у букв
```

Длина последовательности при сравнении не имеет значения

```
print('a' < 'ADC') 
# Вывод в терминал False 

print([1,2,4] > [1,2,3,4]) 
# Вывод в терминал True
```

## ***Наименьшее и наибольшее значение элементов последовательности***

`min()` - метод, для того чтобы найти в последовательностях, содержащих элементы одного типа наименьшее значение
`max()` - метод, для того чтобы найти в последовательностях, содержащих элементы одного типа наибольшее значение

```
my_strinf = "12345" 
print(max(my_string)) 
# Вывод в терминал 5 

my_list = ['abc', 'Abc']
print(min(my_list)) 
# Выод в терминал Abc
```

В данных функциях применяются знаки `>` и `<`

```
my_list = ['123', ['456'], 789] 
print(max(my_list)) 
# Вывод в терминал TypeError: '>' not supported between instances of 'int' and 'str' # Нельзя сравнивать числа со строками (даже если строки притворяются числами).
```
## ***Слияние (конкатенация)***

***Конкатенация*** - объединение двух последовательностей в одну. Выполняется при помощи оператора + и объединение можно производить только последовательностей одного типа

***Результатом будет новый объект:***
```
first_baggage_list = ['Диван', 'Чемодан', 'Саквояж', 'Картина'] 
second_baggage_list = ['Корзина', 'Картонка', 'Маленькая собачонка']
 
full_baggage_list = first_baggage_list + second_baggage_list 

print(full_baggage_list) 
# Вывод в терминал ['Диван', 'Чемодан', 'Саквояж', 'Картина', 'Корзина', 'Картонка', 'Маленькая собачонка']
```

## ***Наличие элемента в последовательности***

Python позволяет узнать, есть ли определенный элемент последовательности. Для этого используется оператор `in` , он возвращает `True` или `False`

```movie_ratings = [4.7, 5.0, 4.3, 3.1] 
print(4.7 in movie_ratings) 
# Вывод в терминал True 

name_movie = 'Джонни Мнемоник' print('ж' inn name_movie) 
# Вывод в терминал True
```

***Так же можно посмотреть и отсутствие элемента в последовательности:***

## ***Срезы***

Из последовательности можно извлечь набор элементов по определенным условиям

***Срез*** - результат, получившийся набор элементов, последовательностью того же типа, что и исходная последовательность 

```
name_mivie = 'Джонни Мнемоник'
print(name_movie[2:10]) # В треминале будут выведены символы со 2 по 9 
# Вывод в терминал онни Мне
```

*В ситуациях, когда конечный индекс указывается явно - элемент с этим индексом не включается в срез* 

![[Pasted image 20260507194233.png]]

Для получения среза можно указать только начальный или конечный индекс элемента, тогда второй границей среза будет конец или начало исходной последовательности 

```
name_movie = 'Джонни Мнемоник'
# Взять сред с 7 элемента и до конца последовательности 
print(name_movie[7:])
# Вывод в терминал: Мнемоник

name_movie = 'Джонни Мнемоник'
# Взять срез от начал последовательности до шестого элемента (не включая шестой)
print(name_movie[:6])
# Вывод в терминал: Джонни
```

Чтобы получить срез из элементов, расположенных с отпрядённым шагом, применяют синтаксис sequence`[start:end:step]`

```
movie = ['Матрица', 'Хакеры', 'Трон', 'Тихушники', 'Сеть']
print(movie[0:5:2])
# Вывод в терминал: ['Матрица', 'Трон', 'Сеть']
```

***Можно получить срез, где последовательность элементов будет инвертирована. Для этого сначала указывается конечный индекс исходной последовательности, потом начальный, а шаг указывается отрицательным числом:*** 

```
movies = ['Матрица', 'Хакеры', 'Трон', 'Тихушники', 'Сеть']
# Шаг должен иметь отрицательное значение 
print(movies[5:0:-1])
# Вывод в терминал: ['Сеть', 'Тихушники', 'Трон', 'Хакеры']

# Можно не указывать границы, а задать только отрицательные значения шага 
print(movies[::-1])
# Вывод в терминал: ['Сеть', 'Тихушники', 'Трон', 'Хакеры', 'Матрица'] 
# Инвертирован весь список
```
