### ***Конструкция if...elif...else***

*Значит, стоит задача определить более двух категорий фильмов: если рейтинг больше 4.7 — прекрасный фильм, если рейтинг меньше 4.7, но больше 3.5 — фильм из разряда «сойдёт», а фильмы с рейтингом ниже 3.5 зачисляем в группу «так себе киношечка».*

***Логика программы:*** 
- если (if) оценка фильма выше 4.7 - выводит реплику фильм крут 
- если (elif) оценка выше 3.5, но не выше 4.7 - смотреть можно 
- иначе (else) вывести строку так себе фильм 
- и напечатать проверка окончена 

![[Pasted image 20260423191335.png]]

- После ключевого слова elif записывается условие 
- После условия ставится двоеточие
- Код в теле блока срабатывает, если условие в elif выполнится 
- Код в теле elif отбивается четырьмя отступами 

*В конструкции `if...elif...else` может быть включено сколько угодно блоков `elif` - при ветвлении это позволит учесть все возможные варианты* 

*Программа читает код ветвления сверху вниз, и как только одно из условий сработает - программа выполнит код в теле этого условия, а в следующие блоки не будет даже заглядывать*

## ***Логический тип bool***

`bool` - логический тип данных, который имеет лишь два значения True (истина) и False (ложь)

***Тип bool*** - это подкласс типа int, и значение True и False можно конвертировать в целые числа: 
- int(True) - приведет к числу 1
- int(False) - приведет к числу 0 
### ***Особенности проверки истинности в логических выражениях:***

- Любое число, не равное 0, или не пустой объект конвертируется в `True`
- Число 0, пустые объекты и значения `None` конвертируются в `False` 
- Сравнение с None выполняется по ключевому слову `is`: оператор `==` не сработает 
## ***Логические операторы***

*Если у фильма нулевой рейтинг — первое условие вернёт `False`, и без вложенных условий `if` обработать такую проверку, вроде бы, невозможно*

***В Python есть возможности объединять логические выражения в одной строке*** - для этого есть логические операторы `and` и `or` . С их помощью можно связать логические выражения в общую систему 

```
if оценка фильма получена [и, одновременно] оценка больше 4.7: 
	print('Отличный фильм, бегом за билетами!')
```
### ***Оператор AND***

Логический оператор `ADN (И)` возвращает `True`, если оба логических выражения вернут `True`. Если хоть одно из выражений вернет `False` - то оператор and вернет `False` 

*Рассмотрим историю программиста, едущего ночью на машине домой в Санкт-Петербурге. У него на пути два моста; каждый мост — это условие: если мост разведён — условие ложно и вернёт `False`, если мост сведён — `True`*

```
if мост №1 сведён AND мост №2 сведён: 
	# Если выражение в if вернёт True 
	Программист доберётся до дома 
else: 
	Программист ночует в офисе
```

### ***Оператор OR***

***Логический оператор `or` («или»)*** - возвращает `True`, если хотя бы одно из логических выражений возвращает `True` 

*Продолжим историю про питерского программиста: он обнаружил два альтернативных маршрута, на каждом из маршрутов — по мосту; теперь его устроит, если сведён хотя бы один из мостов: можно проехать, он доберётся до дома. Если хотя бы один мост вернёт `True` — всё будет `True`*

- Двигатель включен - `status = True`
