Глобальной считается сеть интернет 

***Популярные стандарты:*** 
- **Ethernet 10BASE-T** - со скоростью передачи данных 10 Мбит/с по витой паре 3-й категории и выше 
- **Fast Ethernet 100BASE-T** - со скоростью передачи данных 100 Мбит/с по витой паре 5-й категории 
- **Gigabit Ethernet 1000BASE-T** - со скоростью передачи данных 1000 Мбит/с по витой паре 5-й категории 
- **10 Gigabit Ethernet 10GBASE-T** - со скоростью передачи данных 10 Гбит/с по витой паре 6-й категории 

Биты передаваемые по локальной сети Ethernet, организуются в кадры. Под кадром понимается контейнер, в который помещаются данные и другая служебная информация для передачи по сети. Кадр содержит данные заголовка, конца передачи и фактические передаваемые данные

- **Преамбула** - поле, состоящие из семи байтов, заполненных чередующимися единицами и нулями, которые используются для синхронизации сигналов хопов, обменивающихся данными. Другими словами, преамбула - метка начала кадра, чтобы сетевые устройства могли понять, где заканчивается один кадр и начинается другой 
- **Адрес получателя** - это адрес устройства для которого отправлен данный кадр 
- **Адрес отправителя** - адрес устройства отправивший данный кадр 
- **Тип протокола** - в стандарте Ethernet данное поле содержит код, который идентифицирует протокол сетевого уровня (протокол IP)
- **Данные** - это поле содержит данные, полученные с сетевого уровня передающего компьютера. Затем эти данные передаются протокол компьютера-получателя 
- **Контрольная сумма (FCS)** - контрольная последовательность (сумма) кадра. Это поле включает механизм проверки, позволяющий гарантировать передачу пакетов данных без повреждений (FCS отправки и получателя сравниваются и должны совпадать). Принимающие устройства вычисляет контрольную сумму, и если она не совпадает со значением в FCS отправителя, то кадр уничтожается
