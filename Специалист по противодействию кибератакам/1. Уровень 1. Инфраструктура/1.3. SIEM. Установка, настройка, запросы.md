## ***ПОЛНЫЙ ТЕКСТ ВСЕХ КОМАНД (от начала до конца)***

### ***1. Скачивание и установка публичного ключа Elasticsearch***

```
wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo gpg --dearmor -o /usr/share/keyrings/elasticsearch-keyring.gpg
```
### ***2. Установка пакета apt-transport-https***

```
sudo apt-get install apt-transport-https
```
### ***3. Добавление репозитория Elasticsearch (через зеркало Яндекса, т.к. официальный заблокирован)***

```
sudo rm /etc/apt/sources.list.d/elastic-8.x.list 2>/dev/null
echo "deb [trusted=yes] https://mirror.yandex.ru/mirrors/elastic/8/ stable main" | sudo tee /etc/apt/sources.list.d/elastic-8.x.list
```
### ***4. Обновление списка пакетов и установка Elasticsearch***

```
sudo apt update
sudo apt install elasticsearch -y
```

***Важно:** Во время установки в терминале появится сгенерированный пароль для пользователя `elastic`.*  
***Запомните или запишите его!***  
*В нашем случае это: `WEZJ_Nzygcg0TdsCND1C`*\
### ***5. Запуск Elasticsearch***

```
sudo systemctl daemon-reload
sudo systemctl enable elasticsearch.service
sudo systemctl start elasticsearch.service
```
### ***6. Проверка статуса Elasticsearch***

```
sudo systemctl status elasticsearch.service
```
### ***7. Установка Kibana***

```
sudo apt install kibana -y
```

### ***8. Сброс пароля для служебного пользователя `kibana_system`***

```
sudo /usr/share/elasticsearch/bin/elasticsearch-reset-password -u kibana_system
```

*Система сгенерирует новый пароль. **Запишите его!***  
*В нашем случае использовался: `WEZJ_Nzygcg0TdsCND1C` (но он уникален для каждой установки)*
### ***9. Настройка конфигурационного файла Kibana***

```
sudo nano /etc/kibana/kibana.yml
```

***В самый конец файла добавить следующие строки:***

```
# Мои настройки для подключения к Elasticsearch
server.host: "0.0.0.0"
elasticsearch.hosts: ["https://localhost:9200"]
elasticsearch.username: "kibana_system"
elasticsearch.password: "WEZJ_Nzygcg0TdsCND1C"
elasticsearch.ssl.verificationMode: none
```

***В строке `password` должен быть пароль от `kibana_system` (полученный на шаге 8), а не от `elastic`!***
### ***10. Запуск Kibana***

```
sudo systemctl start kibana
sudo systemctl enable kibana
```
### ***11. Проверка статуса Kibana***

```
sudo systemctl status kibana
```
### ***12. Получение кода верификации (при первом входе)***

```
sudo /usr/share/kibana/bin/kibana-verification-code
```

*На экране появится 6-значный код, например: `123 456`*
### ***13. Вход в веб-интерфейс***

1. Открыть браузер и перейти по адресу:  
    `http://localhost:5601`
2. Ввести код верификации (без пробелов)
3. На странице входа указать:
    - **Username:** `elastic`
    - **Password:** `WEZJ_Nzygcg0TdsCND1C` (пароль с шага 4)

## ПОЛНЫЙ БЛОК КОМАНД ДЛЯ КОПИРОВАНИЯ (всё сразу)

```
# 1. Установка ключа
wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo gpg --dearmor -o /usr/share/keyrings/elasticsearch-keyring.gpg

# 2. apt-transport-https
sudo apt-get install apt-transport-https -y

# 3. Добавление репозитория (зеркало Яндекса)
sudo rm /etc/apt/sources.list.d/elastic-8.x.list 2>/dev/null
echo "deb [trusted=yes] https://mirror.yandex.ru/mirrors/elastic/8/ stable main" | sudo tee /etc/apt/sources.list.d/elastic-8.x.list

# 4. Установка Elasticsearch
sudo apt update
sudo apt install elasticsearch -y

# 5. Запуск Elasticsearch
sudo systemctl daemon-reload
sudo systemctl enable elasticsearch.service
sudo systemctl start elasticsearch.service
sudo systemctl status elasticsearch.service

# 6. Установка Kibana
sudo apt install kibana -y

# 7. Сброс пароля для kibana_system (запомнить вывод!)
sudo /usr/share/elasticsearch/bin/elasticsearch-reset-password -u kibana_system

# 8. Настройка Kibana (открыть файл и вставить строки)
sudo nano /etc/kibana/kibana.yml
# → вставить в конец:
# server.host: "0.0.0.0"
# elasticsearch.hosts: ["https://localhost:9200"]
# elasticsearch.username: "kibana_system"
# elasticsearch.password: "ТВОЙ_ПАРОЛЬ_kibana_system"
# elasticsearch.ssl.verificationMode: none

# 9. Запуск Kibana
sudo systemctl start kibana
sudo systemctl enable kibana
sudo systemctl status kibana

# 10. Получить код верификации
sudo /usr/share/kibana/bin/kibana-verification-code
```
